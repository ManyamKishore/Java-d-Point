What is java

Java is a programming language to communicate , and set of instructions to given machine to do Some task.

      Java is a object-oriented and secure programming language.
      Java was developed by Sun Microsystems (which is now the subsidiary of Oracle) in the year 1995
      James Gosling is known as the father of Java.
      Java works on different platforms (Windows, Mac, Linux, Raspberry Pi, etc.)
      It is open-source and free
      Write once run anywhere (WORA) it was a java slogan

Java is used to develop

  1. Stand alone applications (No need internet)
  2. Web applications (need internet)
  3. Enterprise applications
  4. Desktop applications

Java features

  1. Simple
  2. Object-oriented
  3. Platform independent
  4. Portable
  5. Secured
  6. Multithreading
