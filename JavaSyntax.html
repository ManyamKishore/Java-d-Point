Java Syntax Rules

  1. Every line of code that runs in Java must be inside a class
  2. we named the class name as Main
  3. A class Name should always start with an uppercase first letter.
  4. The name of the java file must match the class name

    For Example : Above class name is Main , file name is Main.java

  5. every program must contain the main() method.
    6. public static void main(String[] args) { }
  6. Note: The curly braces {} marks the beginning and the end of a block of code.
  7. what ever you put inside the println(" ") it will print
    8. System.out.println( ) ;
  8. must end with a semicolon ( ; )
